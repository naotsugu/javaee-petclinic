# Java EE PetClinic Sample Application

## Running petclinic locally

```
	git clone https://github.com/naotsugu/javaee-petclinic.git
	cd javaee-petclinic
	./gradlew war run
```

You can then access petclinic here: http://localhost:8080/petclinic/app
